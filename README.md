# UESTCAutoEvaluator
电子科技大学自动评教（油猴版）

将代码复制到油猴里面新建脚本即可使用

使用方法：

    教务系统->学生评教，然后点击开始评教按钮。
      
    等待一段时间后点击提交即可。

主要逻辑代码来自光爹（kami）
